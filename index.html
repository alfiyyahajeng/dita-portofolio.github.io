Pizza Sales Pivot Dashboard

Pratinjau Laporan: Tren penjualan dan profitabilitas.

This project focuses on analyzing pizza sales data to evaluate overall business performance using descriptive analytics. I built an interactive dashboard in Excel using Pivot Tables and Pivot Charts to identify sales trends, customer demand patterns, and key performance metrics. The dashboard highlights core business insights such as revenue trends over time, best-selling products, and performance by category. This project demonstrates my ability to translate raw transactional data into meaningful insights, apply business metrics, and present them in a structured and decision-oriented format for stakeholders.
